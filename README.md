JOURNAL APP
===

Authors
====
David Odari

Description
---
A journal app to manage notes and ones thoughts.
Built Purely On Firebase

Features
----
* Google Authentication

* Add Notes

* View Notes

* Firebase Authentication

* Swipe To Delete
